6. Summary
The specs/ directory forms the authoritative protocol and governance layer of WLM.
It defines:
- the structural language (SLP)
- the governance model
- the steward’s authority
- the foundation’s boundaries
- the revenue allocation system
All WLM implementations, interpreters, and ecosystem components must align with these specifications.
